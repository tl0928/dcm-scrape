# 05. 交付检查清单

交付前先做一轮检查。

## 文件

- 完整 JSONL。
- 合并后的 CSV。
- 字段说明。
- 失败或缺失列表。
- 采集范围说明。
- 样例说明。

## 数据

- `detail_url` 无重复。
- `name` 不为空。
- `country` 覆盖率可接受。
- `city` 覆盖率可接受。
- `external_website` 缺失时保留空值。
- 坐标缺失时保留空值。
- 结构化字段缺失时保留空值。

## 说明

交付说明里写清楚：

- 数据来自公开页面。
- 字段按页面公开可见内容整理。
- 空值表示页面未公布或未解析到。
- 不猜测、不补全外部来源。
- 每条记录保留 `detail_url`，方便回查。
